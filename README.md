# Taste Vault
AI-Powered Wine Tasting Journal

## 📌 Project Description
TasteVault is a full-stack web application that helps users discover wines, record tasting experiences, and generate professional-style tasting notes using AI.

The app is designed for both casual wine drinkers and enthusiasts who want to better understand and remember their preferences without needing expert-level wine knowledge.

## 🚀 Features
1. 🔍 Wine Search
   - Search from a curated wine database using fuzzy matching.
2. 📝 Tasting Note Creation
   - Record structured tasting data using sliders and tags.
3. 🤖 AI Tasting Note Assistant
   - Generate professional tasting descriptions based on user input.
4. 📚 Wine Journal
   - View and search past tasting note

## 🧱 Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Supabase (PostgreSQL + Auth + RLS) + Express.js + Node.js
- AI Integration: Gemini API
